当然，在今天，我们有了ES6。ES6给我们提供了新的语法Map。Map是针对javaScript中的对象传统上只能用字符串当作键，这样就给使用带来了很大的限制。

Map的构造函数可以接受一个数组作为参数，使用方法如下：

Map的键实际上是跟内存地址绑定的，只要内存地址不一样，就视为两个键。这就解决了同名属性碰撞（clash）的问题，我们扩展别人的库的时候，如果使用对象作为键名，就不用担心自己的属性与原作者的属性同名。

Map具有size属性，set(key,value)方法，get(key)方法。其中set方法设置完键值后会返回该Map所以可以采用链式写法。get方法如果找不到key会返回undefined。另外，还有has(key),delete(key),clear()方法。

Map原生还提供三个遍历器生成函数和一个遍历方法。

  • keys()：返回键名的遍历器。
  • values()：返回键值的遍历器。
  • entries()：返回所有成员的遍历器。
  • forEach()：遍历Map的所有成员。
