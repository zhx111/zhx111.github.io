[leetcode24]Swap Nodes in Pairs(js版)

发表于 2017-06-10 | 分类于 算法 |

笔者最近这段时间都在研究react，所以博客更新的少了一点。还有两个多月就要找工作了，所以又开始刷刷算法题。这次带来的是跟链表相关的问题。

题目要求如下：

Given a linked list, swap every two adjacent nodes and return its head.

For example,
Given 1->2->3->4, you should return the list as 2->1->4->3.

Your algorithm should use only constant space. You may not modify the values in the list, only nodes itself can be changed.

大概翻译一下。

就是将单链表每隔着两个进行交换。同时题目要求不能使用额外的空间。当然，如果链表是奇数个的话，最后一个数字是不用交换的。

笔者一开始的思路是通过循环去做，使用四个指针，两个记录前两个要交换的指针，另外两个则是现有指针往前两个。也就是a，b指向1->2，anext，bnext指向3->4。

当然这样就要考虑如果链表只有1，2，3个的情况。（确实实现的方式非常不优雅，不过笔者还是照这个思路实现了一下）
