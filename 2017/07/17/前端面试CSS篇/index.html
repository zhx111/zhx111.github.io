CSS 2.1（CSS权威指南）

第二章选择器

  • 通配选择器为星号（*）
  • 元素，类，ID选择器
  • 属性选择器
    • 简单属性选择器 h1[class]{color:red;}
    • 根据具体属性值选择 p[class="urgent warning"]{}不同于类选择器，是一个完全串的匹配
    • 根据部分属性值选择
      • [foo~="bar"] 选择foo属性值包含“bar”的所有元素
      • [foo^="bar"] 选择foo属性值以“bar”开头的所有元素
      • [foo$="bar"] 以bar结尾
      • [foo*="bar"] 包含子串“bar”
    • 特定属性选择类型
  • 后代选择器（加一个空格）选择继承的所有元素
  • 选择子元素（使用大于符号>）只是子元素，不包括继承的
  • 选择相邻兄弟元素 （使用+号）只能选择两个相邻兄弟中的第二个元素
  • 伪类和伪元素
    • 链接伪类
      • :link有href属性的超链接
      • :visited作为以访问链接的所有锚
    • 动态伪类
      • :focus当前拥有输入焦点的元素
      • ：hover
      • :active被用户输入激活的元素
    • 伪类顺序 l-v-f-h-a
    • 选择第一个子元素
      • :first-child 常见的错误时认为例如p:first-child之类的选择器会选择p元素的第一个子元素。但实际是作为第一个子元素的p元素
    • 伪元素选择器
      • 设置首字母样式:first-letter
      • 设置第一行样式:first-line
      • 设置之前和之后元素的样式:after :before
