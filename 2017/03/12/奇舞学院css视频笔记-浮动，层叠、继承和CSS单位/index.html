CSS 第二课

浮动

定位模式

  • 常规流(Normal Flow)
  • 浮动(Float)
  • 绝对定位

常规流

  • 根元素、浮动和绝对定位的元素会脱离常规流(out of flow)，其他元素都在常规流之内(in-flow)
  • 常规流中的盒子，属于块级格式化上下文或行级格式化上下文

块级格式化上下文（Block Formatting Context）

  • 盒子在容器内从上到下一个接一个的放置
  • 两个兄弟盒子之间的竖直距离由margin属性决定
  • 同一个BFC内垂直margin会合并
  • 盒子的左外边缘挨着容器的左边

行级格式化上下文（Inline Formatting Context）

  • 盒子一个接一个水平放置
  • 盒子之间的水平margin，border和padding都有效
  • 同一行的盒子所在的矩阵区域叫行盒(Line box)
  • 党一个行盒放不下上下文内所有的盒子时，会被分到多个垂直堆叠的行盒里
  • 行盒内的水平分布由‘text-align’属性决定
  • 如果一个行级块无法分割(单词、inline-block)，该元素会被作为一个整体决定分布在哪一个行盒

浮动(float)

  • 浮动元素从常规流中脱离，被漂浮在容器(包含块)左边或右边
  • 浮动盒会一直漂浮到其外边缘挨到容器边缘或另外的浮动盒
  • 浮动元素不会影响到其后面的流内块级盒
  • 但是浮动元素后面的行级盒子会变短以避开浮动元素

clear

  • 指定元素哪一边不能与之前的浮动框相邻
  • 取值：left | right |both

clearfix

加到浮动元素的父级元素

块级格式化上下文（BFC）的特性

  • BFC内的浮动不会影响到BFC外部的元素
  • BFC的高度会包含其内的浮动元素
  • BFC不会和浮动元素重叠
  • BFC可以通过overflow：hidden等方法创建

BFC的创建

  • 浮动框
  • 绝对定位框
  • 非块级的块容器(inline-block)
  • overflow属性非visible的块框

BFC的作用

  • 清除浮动
  • 防止margin折叠
  • 双栏布局

属性的覆盖

属性的覆盖可以通过增加一个类，在那个类中修改所需要的属性。

Important

!important 提高优先级。谨慎使用！

Cascading

css样式的来源

  • 页面开发者
  • 用户设置
  • 浏览器预设

哪条声明起作用

  1. 找出匹配到的该属性所有声明
  2. 根据规则来源，优先级从低到高：
    • 浏览器预设
    • 用户设置
    • 网页样式
    • （含!important的网页样式）
    • （含!important的用户设置样式）
  3. 同一来源中，按照特异度排序，越特殊优先级越高
  4. 特异度一样时，按照样式书写顺序，后面的优先级高

默认值策略

继承

某些属性会自动继承其父元素的计算值，除非显示指定一个值

显式继承

例如：box-sizing: inherit;

初始值

  • css中，每个属性都有一个初始值
  • background-color的初始值为transparent
  • margin-left的初始值为0
  • 可以显示重置为初始值，比如background-color:initial

CSS求值过程

DOM树+样式规则=>filtering=>cascading=>defaulting=>resolving(将一些相对值或者关键字转化为绝对值)=>formatting(将计算值进一步转换)=>constraining（将小数像素转为整数）=>实际值

各种类型的值

  • 关键字：initial、inherit
  • 字符串:content:”abc”
  • url(path..)
  • 长度：em, px, vm..
  • 百分比
  • 整数
  • 浮点数
  • 颜色：#fff、rgba()
  • 时间：transition-duration:0.3s
  • 角度：
  • 函数：

相对单位

  • em：相对于该元素的一个font-size
  • rem：相对于html元素的font-size
  • vh：浏览器窗口的1%
  • vw：浏览器窗口宽度的1%
  • vmin：vh和vm中的较小者
  • vmax：vh和vm中的较大者

颜色

  • 关键字
  • Hex
  • RGB&RGBA
  • HSL&HSLA

rgba中的a为透明度，和opacity相比的好处是，它不会造成对子元素透明。

HSL的意义：

Hue：色相(H)是色彩的基本属性，就是平常所说的颜色。取值范围：0-360

Saturation：饱和度(S)是指色彩的纯度，越高色彩越纯，低则逐渐变灰。取值范围0-100%

Lightness：越高颜色越亮。取值范围是0-100%。

例如：hsl(0,50%,50%),hsla(120,50%,30%,0.5)
