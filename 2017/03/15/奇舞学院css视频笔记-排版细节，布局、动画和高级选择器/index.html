CSS第六课

排版细节

行级格式化上下文

font Metrics

line-height是两行之间baseline的高度差。

Vertical-align

  • 定义盒子所处的行盒(line box)的垂直对齐关系
  • 取值：baseline | sub | super | top | text-top | middle | bottom | text-bottom | \ | \
  • 百分比相对于元素自身的行高
  • 初始值为baseline

lists style

  • display:list-item会生成Principle Block Box和Marker Box
  • Marker Box的内容和位置可以通过list-style系列属性指定

（ul中li元素前面的小点就是Marker Box）

list-style-position

  • inside
  • outside

list-style-type

  • none
  • disc
  • circle
  • square
  • decimal
  • lower-roman
  • upper-roman
  • lower-greek
  • …

list-style-image

url(…)

以上属性可以缩写为list-style

background

  • background-color
  • background-image
  • background-repeat
  • background-position

background：red url() no-repeat 0 0;

CSS Sprites

利用position属性，使用图片的一小部分

background-size

css新增

background-size：200px 100px；

contain：让容器整个包含图片，同时尽量不裁剪

cover：用图覆盖容器，尽量填满而不管是否裁剪图片

border-radius

  • 可以指定四个方向
  • 可以指定百分数

box-shadow

box-shadow:0 0 0 3px rgba(..)

5个位置分别是：

  1. 阴影的水平距离
  2. 垂直距离
  3. 模糊半径
  4. 伸展半径
  5. 颜色（透明度）
  6. inset内阴影

box-shadow可以写多个

CSS第七课

布局

布局方法

  • float
  • position
  • inline-block
  • table
  • flex
  • grid

居中

水平居中

  • 行级元素：text-align:center
  • 块级元素:margin:auto

垂直居中

  • 单行文字：line-height
  • 行级盒子：vertical-align：middle
  • 绝对定位：top:50%；left:50%（transform：translate(-50%,-50%)）

float-based layout

利用float布局

float的时候利用padding赋予空间

用负margin移动位置

用position：relative将float元素移到正确位置

伪等高

padding-bottom:99em;margin-bottom:-99em

两个元素浮动，利用padding-bottom撑开较小的盒子，在用margin负值将盒子实际的高度减回来。

同时父元素需要overflow:hidden

position

利用position:relative和absolute布局

table layout

表格布局通常只用于表格，不要使用表格标签布局

table的宽度自动计算，占满屏幕要100%

border-collapse:collaspse将相邻的border合并

table-layout:fixed，表格列宽度相等

display：table

  • display：table
  • ～：table-cell
  • ～：table-row
  • ～：table-column
  • ～：table-columen-group
  • ～：table-footer-group
  • ～：table-header-group

应用：

父元素用display:table

子元素用display:table-cell

实际上是形成了一个表格

hsl颜色属性：控制hs调节l 可得到相同色系的

Flexbox

真正为解决布局问题而生的规范

flexbox可控制子元素

  • 水平或垂直成一行
  • 控制子元素对齐方式
  • 控制子元素的宽度／高度
  • 控制子元素显示顺序
  • 控制子元素是否折行

display:flex

  • 将元素变成flexbox容器
  • 元素从左到右或从上到下顺序排布

flex-direction

  • 子元素排列方向
  • 取值：row | row-reverse | column | column-reverse

flex-grow

  • 定义每一个子元素在盒子内的弹性
  • 拓展盒子剩余空间的能力
  • 取值：默认0
  • 子元素属性

flex-shrink

  • 元素收缩的能力
  • 取值：默认1

flex-wrap

  • 元素在主轴方向排放时，能否换行
  • 取值：wrap | nowrap | ..

justify-content

  • 子元素沿主轴方向的摆放
  • 取值：flex-start | flex-end | center | space-between | space-around

align-items

  • 在侧轴方向的对齐方式
  • 取值：flex-start | flex-end | center | baseline | stretch
  • 默认为stretch

align-self

  • 子元素设置的属性

align-content

  • 多行内容在容器内侧轴的对齐方式

order（重要）

  • 制定摆放时的顺序，从小到大
  • 取值：默认0

CSS 第八课

动画

transform 变换

  • 对元素进行平移、旋转、缩放
  • transofrm不会对其他元素布局产生影响
  • 取值：none | \

Transform-list:

translate(x,y)

rotate(xdeg)

scale(2,0.5)沿x轴和y轴

可以指定多个变化，上面的值通过空格隔开

transform书写顺序会影响显示效果

指定某一坐标轴变换

  • translateX
  • translateY
  • scaleX
  • scaleY

transform-origin

旋转或缩放的中心点

默认为元素的中心点旋转/缩放，可以通过transform-origin指定

3D Transform

  • 指定进行3D渲染时，人眼距离Z平面的距离
  • 不影响元素本身的渲染
  • 只影响子元素的3D渲染

perspective:10em越大离眼睛越远

  • translate3d()
  • translateZ()
  • rotate3d()
  • rotateX()
  • rotateY()

transition 过渡

  • 指定从一个样式状态到另一个状态时如何过渡
  • 动画的意义：告诉用户发生了什么

指定过渡

  • 什么属性发生变化需要过渡
  • 过渡持续多长时间
  • 速度变化时什么样
  • 是否有延迟

transition

  • transition-property
  • transition-duration
  • transition-timing-function（线性变化等）
  • transition-delay（延迟）

transition-timing-function：

贝塞尔曲线

ease | linear | ease-in | ease-out | ease-in-out

cubic-bezier(.24,.63,.67,.62)

可以造成回弹的效果

animation

  • animation可以实现更复杂的样式变化效果
  • 定义关键帧
  • 指定动画行为

@keyframes为定义关键帧 down为动画名

from 50% to为三个状态

声明完动画后可以通过animation属性定义

animation: down 1.5s ease infinite;

CSS 第九课

高级选择器 Advanced Selector

:target 伪类

  • 表示元素被hash匹配时的状态
  • 比如URL是/post/a#heading时。name为heading的元素处于被target的状态

:lang 伪类

  • 元素匹配上指定语言时的状态
  • 浏览器通过lang属性获得语言信息

:nth-child

  • 通过:nth-child(an+b)选中某些子元素
  • 例如:nth-child(3n)选中第3、6、9……个子元素
  • a可以为负数

:nth-of-type

只对相同类型的元素起作用

:first-child/:last-child

第一个/最后一个子元素起作用

比如第一个元素对前面的元素没间距

:not()

  • 排除匹配的元素
  • 比如img:not([alt])选择没有写alt属性的图片

button:not(:last-child){}

其他选择器

  • :nth-last-child()
  • :nth-last-of-type()
  • :first-of-type
  • :last-of-type
  • :only-child
  • :only-of-type

:empty

  • 匹配中间没有内容的标签

::first-line（伪元素）

p标签的第一行颜色为红色

::first-letter

改变第一个字母的样式

兄弟选择器

  • 相邻兄弟选择器E+F
  • 通用兄弟选择器E~F

+表示在后面紧挨的第一个兄弟节点（在前面的不会）

～表示在后面紧挨的所有兄弟节点

:checked ~ F

:check伪类会选中点击的checkbox或者radio
