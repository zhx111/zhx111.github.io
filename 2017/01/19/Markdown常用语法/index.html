什么是Markdown

来自维基百科：

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。[4]这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。

Markdown is a text-to-HTML conversion tool for web writers. Markdown allows you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid XHTML (or HTML).

简单的说，markdown是一种语法，可以让你在写文章的时候就把格式也确定下来，而像我们常用的word就是先写内容，再进行格式的编辑。

用处

markdown在各种地方运用的越来越广泛，Github的README.md，包括本博客的每篇文章，都是采用markdown语法写的。

markdown的语法很简单，接下来，本文将介绍一些常用的markdown语法。
