用Hexo+Github pages搭建自己的博客

发表于 2017-01-08 |

前言

一直很想建一个属于自己的个人博客，在经过一番筛选之后，我选择了Hexo+Github Pages这个组合来搭建我的博客。选用这个的好处是因为Github是每个程序员必备的，而Github提供了一个可供展示自己静态网站的稳定空间，我们可以利用这个建立自己博客。而Hexo正如它的首页所说，是一个快速、简洁而且高效的博客框架。

建站过程

下面介绍了本博客的建站过程，想得到更多详细的信息，请查看hexo的官网：https://hexo.io/docs/index.html

另外本人使用系统为macOS

建站环境配置

安装Hexo的过程非常简单，你只需要在电脑中拥有：

  • Node.js
  • Git

Node.js是一个基于Chrome V8引擎的javascript运行环境，其提供的npm是目前最大的开源包管理系统；Git则是一个现在非常流行分布式项目管理版本控制系统，我们熟知的Github就是基于git的最大的开源项目托管平台（这两个应该都非常熟悉了）
