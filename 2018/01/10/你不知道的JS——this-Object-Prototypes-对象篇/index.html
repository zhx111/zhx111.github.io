Chapter3:Object

语法

对象的两种形式：声明（字面）形式和构造形式。

大多数情况下都使用字面形式。

类型

对象是JS六种主要类型中的一种：

  • string
  • number
  • boolean
  • null
  • undefined
  • object

除了object是简单几本类型，它们自身不是object。

null有时会被当成一个对象类型，但是这是由于typeof null返回object造成的，这是个bug。null是它自己的几本类型。

内建对象

  • String
  • Number
  • Object
  • Function
  • Array
  • Date
  • RegExp
  • Error

在JS中，它们实际上仅仅是内建的函数。这些内建函数的每一个都可以被用作构造器。

在必要的时候，语言会自动将string类型强制转位String对象类型（比如调用字符串的方法），所以JS社区绝大部分人强烈推荐尽可能使用字面形式的值。

内容

对象的内容由存储在特定命名的位置上的（任意类型的）值组成，这些值称为属性。

这些内容在实现上并不存储在对象内部。对象中实际存储的是值的引用(reference)

在对象中，属性名总是字符串。使用其它值会被转位字符串。

属性描述符(Property Descriptors)

ES5中，所有属性都用属性描述符来描述。

我们可以通过Object.defineProperty(...)来添加新属性。

可写性（Writable）

这个属性控制你改变属性值的能力。

在非严格模式下会修改失败，严格模式下会抛出错误。

可配置性（Configurable）

只要属性当前是可配置的，我们就可以使用相同的 defineProperty(..) 工具，修改它的描述符定义。

configurable:false 阻止的另外一个事情是使用 delete 操作符移除既存属性的能力。

可枚举性（Enumerable）

这个性质控制一个属性是否能在特定的对象-属性枚举操作中出现。

不可变性

对象常量

通过将writable:false 与 configurable:false 组合，你创造了一个不能改变重定义或删除的变量。

防止扩展

如果你想防止一个对象被添加新的属性，但同时需要保留其它既存的对象属性，可以调用Object.preventExtensions(..)

封印(Seal)

在当前对象上调用Object.preventExtensions(..)，并且将它所有的既存属性标记为configurable:false。

第五章：原型

[[Prototype]]

JavaScript中的对象有一个内部属性，在语言规范中称为 [[Prototype]]，它只是一个其他对象的引用。几乎所有的对象在被创建时，它的这个属性都被赋予了一个非 null 值。

在之前提到的[[Get]]操作中，如果查找的某个属性在对象上不存在的话，就会从[[Prototype]]链上继续处理。

这种查找会遍历对象的整个原型链。

object.prototype

每个普通原型链的最顶端，是内建的object.prototype

设置与遮蔽属性

观察以上代码，如果属性名foo同时存在于myObject本身和从myObject开始的原型链的更高层，这样的情况称为遮蔽。直接存在于myObject上的foo属性会遮蔽任何出现在链条高层的foo属性。

当 foo 不直接存在 于 myObject，但 存在 于 myObject 的 [[Prototype]] 链的更高层时：

  1. 如果一个普通的名为 foo 的数据访问属性在 [[Prototype]] 链的高层某处被找到，而且没有被标记为只读，那么一个名为 foo 的新属性就直接添加到 myObject 上，形成一个 遮蔽属性。
  2. 如果一个 foo 在 [[Prototype]] 链的高层某处被找到，但是它被标记为 只读（writable:false） ，那么设置既存属性和在 myObject 上创建遮蔽属性都是 不允许 的。如果代码运行在 strict mode 下，一个错误会被抛出。否则，这个设置属性值的操作会被无声地忽略。不论怎样，没有发生遮蔽。
  3. 如果一个 foo 在 [[Prototype]] 链的高层某处被找到，而且它是一个 setter（见第三章），那么这个 setter 总是被调用。没有 foo 会被添加到（也就是遮蔽在）myObject 上，这个 foo setter 也不会被重定义。

如果你想在第二和第三种情况中遮蔽 foo，那你就不能使用 = 赋值，而必须使用 Object.defineProperty(..)将 foo 添加到 myObject。
